# Domain 2: Design and Implement a Source Control Strategy (10–15%)

This domain covers branching strategies, PR workflows, repository management, and Git operations.

## Exercises

| # | Exercise | Skills Covered | Difficulty |
|---|----------|---------------|------------|
| 2.1 | Branching Strategies | Trunk-based, feature branch, release branch | Beginner |
| 2.2 | Pull Request Workflows & Branch Policies | PR flows, branch protections, merge restrictions | Intermediate |
| 2.3 | Managing Large Files with Git LFS | Git LFS, git-fat, large repositories | Intermediate |
| 2.4 | Repository Scaling & Optimization | Scalar, shallow clones, sparse checkout | Intermediate |
| 2.5 | Repository Permissions & Tags | Permissions, tags, CODEOWNERS | Beginner |
| 2.6 | Git Recovery & Data Operations | Git recovery, removing sensitive data, reflog | Advanced |
